При вызове класса SS "$SS = new ss($var);" - передаётся подключение к базе данных.

* Метод ss_String - экранирует спец. символы в строковой переменной.
    Входные данные - строка.
    Выходные данные - обработанная строка.

* Метод ss_Int - экранирует спец. символы в числовом значение.
    Входные данные - число или строка.
    Выходные данные - обработанная строка с числовым значением.

* Метод ss_Int_mm - экранирует спец. символы и проверяет число, находиться ли оно в указанном диапозоне. Если нет, то подставляет значение по умолчанию.
    Входные данные - 1) Число или строка 2) минимальное значение 3) максимальное значение 4) Значение по умолчанию.
    Выходные данные - обработанная строка с числовым значением.

* Метод ss_HTML - используется если полученные данные от пользователя необходимо отображать на странице. Обрабатывает HTML тэги, чтобы отображались как текст.
    Входные данные - строка.
    Выходные данные - обработанная строка.

* Метод ss_Trans - используется для перевод кириллицы в транслит.
    Входные данные - строка.
    Выходные данные - строка в транслите.
